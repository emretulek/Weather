# Weather Widget Plugin

This project was developed as a weather widget for windows widgets.

### Internal Settings

**reload_time** time unit seconds

```json
{
  "reload_time": 600,
  "font_big": 24.0,
  "font_medium": 16.0,
  "font_small": 12.0,
  "color_light": "#FFFFFFFF",
  "color_dark": "#FF808080",
  "address": "İstanbul, Türkiye",
  "geocode": "41.009,28.966"
}
```
